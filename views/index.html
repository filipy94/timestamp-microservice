Timestamp Microservice

Build a full stack JavaScript app that is functionally similar to this: FCC Timestamp Microservice Model.

Example URL path:

  • [your project url]/api/2015-12-25 - with a valid date format
  • OR
  • [your project url]/api/1451001600000 - with a valid unix format

Example Output:

{"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}
